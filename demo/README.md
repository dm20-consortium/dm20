# demo_env

DMの分散配置の一例のデモンストレーションを提供します

## 動作確認環境

Ubuntu 20.04
Docker version 24.0.5, build 24.0.5-0ubuntu1~20.04.1

## デモ概要

- ２つのエッジサーバで観測した物標情報をクラウドサーバに集約し，それを車両へ伝えるデモです．

## 使用方法

- DMのDockerイメージをビルドします

```bash
  cd ..
  ./build.bash
```

- 必要な環境変数を設定します

```bash
  cd demo
  export DEMO_SAMPLE_PATH=`pwd`
```

- dockerを実行します

```bash
  docker compose up --force-recreate
```
