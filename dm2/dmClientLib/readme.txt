/************************************/
　　ライブラリの生成・ビルド方法
/************************************/

○ライブラリの生成
　基本はmakefileに記載されているため
　makeコマンドでbin配下に共有ライブラリが生成される。

○ビルド方法
　実際にライブラリを使用する場合は以下の手順を実施する

　①本ソースのinclude配下のソースを/usr/local/include/is配下にコピーする
　　※/usr/local/include/isフォルダがない場合はmkdirすること

　②生成したライブラリを/usr/local/lib配下にコピーする

　③ライブラリを使用するアプリをビルドする際に-lisのオプションを付与してビルドする

　　本ソース群に含まれているmain.cppにて動作確認する場合は以下でビルドを実施
　　g++ -std=c++14 main.cpp -lis -lcrypto -lssl -o main.out

○Dockerイメージのビルド方法

  ①ビルドコマンドの実行

    docker build -t IMAGE_NAME:TAG_NAME .

    IMAGE_NAME: イメージ名を指定する．任意であるが，わかりやすさのためislibと指定することを推奨する 例：islib
    TAG_NAME  : タグ名を指定する．任意であるが，SVNのリビジョン番号を含めることを推奨する．         例：r1495
