*********************************
   ユーザ定義関数作成について
*********************************

以下の手順を実施する事でユーザ定義関数をSQL発行できるようになる。


■ユーザ定義関数が定義された共有ライブラリを生成する

①同ディレクトリに格納されている"sample.cpp"を参考に関数を実装する

②以下、コマンドを実行し、共有ライブリを生成する

// 共有ライブラリの生成
g++ -std=c++14 -shared -fPIC -o XXXXX.so XXXXX.cpp

③lib/eval/配下にXXXXX.soファイルを配置する


■設定ファイルに定義する

①conf/evalFunction.confを開く

②以下の内容を追記する

関数名_LIB=XXXXX.so
関数名_ARG_TYPE=引数1の型,引数2の型,引数Nの型
関数名_RET_TYPE=戻り値1の型,戻り値2の型,戻り値Nの型
関数名_RET_NAME=戻り値1のカラム名,戻り値2のカラム名,戻り値Nのカラム名
※4つ目のRET_NAMEは戻り値が1つであれば省略可
